## Support

If you need assistance using the SAP ECC Customer System API, contact the integration platform support team at [support@brilliantmule.com](mailto:support@brilliantmule.com).

Support hours are Monday through Friday from 9:00 AM to 5:00 PM Pacific Time.

The target response time for support inquiries is within four business hours.

## Before Requesting Support

Before submitting a support request, please review the following resources:

- The documentation pages of this asset.
- The API specification and resource documentation.
- The API Response Traits Library, API Security Schemes Library, and Common Response Types Library.

These resources address most integration and usage questions.

## Information to Include

When submitting a support request, please include the following information:

- Exchange asset name and version.
- Name of the client application consuming the API.
- Description of the issue or question.
- Relevant error messages or logs.
- A sample request or payload when applicable.

## Intended Usage

The SAP ECC Customer System API provides access to customer data stored in SAP ECC. Support requests should be related to:

- Issues invoking API endpoints.
- Unexpected responses or errors.
- Questions regarding request or response structures.
- Access or connectivity issues.
